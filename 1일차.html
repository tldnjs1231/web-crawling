web¶

  • reqeust: client가 server에게 자원을 요청하는 것
  • response: client가 요청한 자원을 server가 응답하는 것

client가 server에게 자원을 요청하는 단계¶

  1. web page 요청 -> request
  2. 요청한 파일을 검색
  3. 요청한 파일 전달 -> response
  4. web browser가 파일 처리
